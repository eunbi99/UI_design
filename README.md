# VGood - 비건을 위한 웹사이트 
### 🥗프로젝트 주제 
> 전 세계적으로 건강과 환경, 윤리적 소비를 중시하는 소비자들이 늘면서 '비건 라이프' 지향하는 사람들이 증가하고 있으며,
>  소수의 취향으로 여겨왔던 ‘비건(vegan)’이 우리나라에서도 주목을 받고 있다. 이러한 비건에 대한 소개와 비건 음식에 대한 구독 서비스를 가진 웹 사이트 UI를 만들고자 하였다.
>> ### ⏰프로젝트 기간
>  기간 : 2021.04.26 ~ 2021.04.29
>> ### 💻사용 기술 
> - BootStrap
> - HTML
> - CSS
> - jQuery
>> ### 💻맡은 역할 
> - 비건이야기
> - 후원하기 
> - 우측 하단 top, 챗봇
